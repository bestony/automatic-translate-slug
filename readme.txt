Make Your Post Slug Into English

== Description ==
This Plugin use Some Translate Provider to Translate Your Post Title into English as Post Slug, Then, Your Slug Didn\'t have a URL Encode String (like \"%9e\").

== Installation ==
1. Download Plugin
2. Install Plugin
3. enter The Plugin Page, and Choose Service Provider,
4. Input Appid and Secret.
5. Save Settings.

== Frequently Asked Questions ==
= Where Can I get Key =
For YouDao Fanyi : You can get Key from http://fanyi.youdao.com/openapi?path=data-mode
For YouDao AI: You can get Key from https://ai.youdao.com
